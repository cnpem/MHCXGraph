Installation
============

To install the latest release from `PyPI <https://pypi.org/project/MHCXGraph>`_, 
run:

.. code-block:: bash

  pip install MHCXGraph

Or to install the latest developmental version from source, run:

.. code-block:: bash

  git clone https://github.com/CNPEM/MHCXGraph.git
  pip install -e MHCXGraph
